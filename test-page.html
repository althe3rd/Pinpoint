Debugging Tips

  • Open Developer Tools: Press F12 and check the Console tab for detailed results
  • Check Network Tab: Make sure axe-core loads successfully from CDN
  • Try a Simple Test: Type console.log('test') in the console to ensure JavaScript is working
  • Internet Connection: The bookmarklet needs to download axe-core from CDN

Expected Results

When you run the bookmarklet on this page, you should see:

  • Multiple violations detected (4-6 issues)
  • Some items marked for manual review
  • Detailed results in the browser console

If you see these results, the bookmarklet is working correctly!
