Privacy Policy

Last updated: November 14, 2024

TL;DR: Pinpoint Accessibility Checker processes all data 100% locally in your browser. We don't collect, store, or transmit any of your personal information, website content, or accessibility testing results. All code runs entirely offline with no external connections.

1. Introduction

This Privacy Policy describes how Pinpoint Accessibility Checker ("we," "our," or "the extension") handles information when you use our browser extension. We are committed to protecting your privacy and being transparent about our data practices.

2. Information We Do NOT Collect

Pinpoint Accessibility Checker is designed with privacy as a core principle. We do not collect, store, or transmit:

  • Personal Information: No names, email addresses, or personal identifiers
  • Website Content: No HTML, text, images, or other content from websites you test
  • Accessibility Results: No testing results, scores, or violation details
  • Browsing History: No record of which websites you visit or test
  • Usage Analytics: No tracking of how often you use the extension
  • Device Information: No information about your computer or browser

3. Local Data Storage

The extension stores the following data locally in your browser only:

User Preferences

  • Filter settings (which types of accessibility issues to display)
  • Panel position and size preferences
  • WCAG compliance level settings (AA vs AAA)
  • Custom exclude selectors for testing
  • Manual review verification states

Important: This data never leaves your browser and is automatically deleted if you uninstall the extension or clear your browser data.

4. Third-Party Libraries

axe-core Library (Bundled)

The extension includes the axe-core accessibility testing library (v4.10.3) bundled directly within the extension package:

  • Included: axe-core v4.10.3 by Deque Systems
  • Purpose: Provides the accessibility testing engine
  • Location: Bundled locally with the extension (no external loading)
  • Data Shared: None - runs entirely offline in your browser
  • Network Connections: Zero - no communication with external servers

Note: As of version 1.5.76+, the extension is fully self-contained with all dependencies bundled locally to comply with Manifest V3 requirements and maximize user privacy.

5. Permissions Explanation

The extension requests the following browser permissions:

activeTab

Why needed: Allows the extension to run accessibility tests on the currently active tab when you click the extension icon.

Data access: Reads page content to analyze accessibility issues.

Privacy impact: Content is analyzed locally and never transmitted.

scripting

Why needed: Injects the accessibility testing scripts into web pages.

Data access: Executes analysis code on the current page.

Privacy impact: All processing happens in your browser only.

contextMenus

Why needed: Adds a right-click menu option to run accessibility tests.

Data access: None - only adds menu interface.

Privacy impact: Zero - purely for user convenience.

commands

Why needed: Enables keyboard shortcuts (Ctrl+Shift+A / Cmd+Shift+A).

Data access: None - only keyboard event handling.

Privacy impact: Zero - purely for accessibility features.

6. Data Security

Since all processing happens locally in your browser with no external connections:

  • Zero network requests - Extension operates completely offline
  • No data transmission - Nothing sent to external servers
  • No tracking or analytics - We don't know you're using the extension
  • No risk of data breaches - Your data never leaves your device
  • Complete control - You own all data on your local machine
  • No accounts required - No logins, registrations, or authentication
  • Manifest V3 compliant - Follows strictest browser security standards

7. Children's Privacy

The extension does not collect any personal information from users of any age. Since we don't collect data, there are no special considerations for users under 13.

8. International Users

The extension works the same way for users worldwide. Since all data processing is local, there are no international data transfer concerns.
